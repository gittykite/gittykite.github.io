 # 한국사 고급 (조선후기~개항기)

# 조선 후기

## 조선후기의 정치
* 비변사, 5군영, 속오군
* 예송, 환국
* 영조, 정조
* 세도정치, 삼정의 문란, 홍경래의 난, 임술 농민 봉기
* 영정법, 대동법, 균역법
* 광작, 도조법, 선대제, 덕대
* 공인, 통공 정책, 송상, 상평통보
* 공명첩, 잔반, 소청 운동, 노비종모법, 향전

## 조선후기의 사회
* 실학, 균전론, 한전론, 여전론
* 소비론, 홍대용
* 양명학, 국학
* 곤여만국전도, 거중기
* 한글소설, 양반전, 청화백자
* 법주사 팔상전, 진경산수화, 풍속화, 민화
* 정감록, 서학, 동학 

# 개항기

## 흥선대원군 ~ 갑신정변
* 비변사 폐지, 경복궁 중건, 서원 철폐, 호포제, 사창제
* 병인박해, 병인양요, 오페르트 도굴 미수 사건
* 제너럴 셔먼호 사건, 신미양요
* 강화도 조약, 최익현
* 통리기무아문, 박문국, 조선책략, 조사 시찰단, 보빙사
* 위정척사 운동, 임오군란, 제물포 조약
* 개화파, 갑신정변, 톈진 조약, 지조법 개혁
* 영국 거문도 점령

### 흥선 대원군의 개혁정치
+ 비변사 폐지  
:  의금부(행정) + 삼군부 부활 => 세도가문(안동김씨) 타파
+ 법규정비  
	- 대전회통 (경국대전 + 속대전)  
	- 육전조례 (행정법규+사례집) 
+ 경복궁 중건
	- 원납전 강제징수
	- 당백전 발행 (상평통보 * 100) => 인플레이션
	- 부역 동원
	- 양반 묘지림 벌목   
+ 서원철폐  
	- 47개소만 남기고 철폐  
	=> 지방 권력(붕당 근거지) 약화, 국가 재정 확충  ex) 만동묘 철폐 (명나라 황제 신종) 
+ 경제정책 (삼정문란 시정)

	|삼정|개혁|내용|
	|---|---|---|
	|전정 | 양전사업 | 토지조사사업 (면세지 색출)|
	|군정 | 호포법 |  호마다 포 제출 (양반 포함)|
	|환곡 | 사창제 | 민간인에게 빈민구휼창고 관리를 위임<br>  =>공무원의 이자착취 방지|

	** 운현궁에 거주 (고종 즉위 후 궁 칭호 받은 개인저택)  
	** 진실로 백성에게 해되는 것이 있으면 비록 공자가 다시 살아난다 하더라도 나는 용서하지 않겠다.

### 통상수교 거부 정책 (쇄국)
+ 병인박해 (1866)    
    : 프랑스로 러시아 견제하려다 유생들 반대로 천주교 박해 
+ 제너럴 셔먼호 사건 (1866)  
    : 평양에서 통상 요구 및 난동 => 박규수가 불태움 
+ 병인양요 (1866) 
    - 프랑스의 강화도 공격    
      * 한성근(문수산성)
      * 양헌수(정족산성)   
    - 외규장각 약탈  
      ex) 직지심체요절, 왕오천축국전
+ 독일 오페르트 도굴 미수 사건 (1868년)  
    - 남연군(흥선대원군 아버지) 묘 도굴 시도 + 통상요구   
    - 실패: 다층 석회 + 주민 저항(충청도 덕산)
+ 신미양요 (1871년)  
    - 미국의 강화도 공격  
    - 어재연 장군 (초지진, 덕진진, 광성보)   
    - 척화비 건립  
     * 서양에 대한 자신감 고조
     * 통상수교 거부 의지   
     * '서양 오랑캐들과 사이좋게 지내자고 주장하는 것은 매국하자는 것이다'

### 해외 사절단 
  - 일본 조사 시찰단 (1881) 
    * 박정양, 홍영식, 유길준(유학) 등
    * 암행어사 형태로 비공개 파견 (유생 등 반발 우려) => 국왕에게 보고서 작성
  - 일본 수신사 (1876)
    * 1차: 김기수 등, 일본 관청/군사학교 시찰
    * 2차: 김홍집 등, 조선책략(친중국/결일본/연미국 조언) 입수
  - 청 영선사 (1881-82)
    * 김윤식 등 톈진 파견
    * 과학/무기 기술 및 외국어 학습 -> 중도 귀국(비용 부족) -> 기기창 설립
  - 미국 보빙사 (1883)
    * 전권대신 민영익, 홍영식, 서광범 등
    * 유길준은 그대로 유학 => 서유견문 집필
    * 근대 시설 시찰 -> 우정국 / 육영공원 설치
    * 일화: 아서 대통령 접견 시 엎드려 절함

    |항목|내용|

    -  14개조 개혁 정강 
      * 내각 중심 입헌군주제
      * 재정의 호조 일원화
      * 규장각 혁파
      * 신분제 폐지 
      * 지조법(토지세금) 개혁
      * 혜상공국(보부상 관리) 폐지  
    - 한계
      * 외세 의존
      * 토지개혁 소홀
      * 민중 지지 X

### 갑오/을미개혁
+ 제1차 갑오개혁 (1894년)  
: 일본이 설치한 군국기무처(정치/군사 기구)에서 실시  
	- 정치
		* 1차 내각: 온건 개화파 김홍집 중심
		* 개국 기년 사용: 1392년 조선 건국  (청 연호 폐지)
		* 궁내부 설치: 왕실 사무 관리 (왕실-국가 분리)
		* 의정부 6조 -> 의정부 8아문
	- 경제
		* 도량형 통일
		* 재정의 탁지아문 일원화
		* 조세 금납화
		* 은 본위제도
	- 사회
		* 공/사 노비법 혁파 -> 신분제 폐지
		* 연좌제 폐지
		* 조혼 금지
		* 과부 재가 허용

### 아관파천 (1896년)
+  명성왕후 시해 후, 러시아 공사관으로 이동한 고종이 일본 주도 개혁 무효화
	- 단발령 비강제화
	-  김홍집 처단(갑오/을미개혁 진행)
+  삼림 채벌권 요구 => 최혜국 대우 적용 확장
	-  미국: 운산 금강 채굴권 요구
	-  프랑스: 광산채굴권 요구
+ 부산 절영도 조차 요구 
	- 석탄 공급 용 부동항 
	- 독립협회 만민 공동회가 무산 시킴

### 독립협회
: 서재필/이상재 등 독립신문 간행을 위해 모인 사람들이 설립 (1896년부터 활동 활발)
+ 고종 환궁 주장
+ 의회설치 주장
+ 독립문 건설 
	- 영은문 자리에 건립
	-  청나라로부터 독립 다짐
+ 민중 계몽 활동
	- 토론회, 강연회
+ 정치 논의 집회
	- 만민공동회: 종로 종각 앞 연설 & 논의   
	  => 절영도 조차 무산, 한/러 은행 철수, 프랑스 광산 채굴권 방어
	- 관민 공동회 : 정부 관료도 참여 (박정양 내각~)

### 대한국 국제 (1899년)
: 황제의 천상천하 유아독존 최고권력 선포한 헌법의 일종
+ 제2조
  - 대한 제국의 정치는 이전부터 이후까지 변하지 않을 전제 정치이다
+ 원수부 설치
  - 모든 군사력 황제에게 일원화
+ 광무개혁 본격화
  - 상공업 진흥
  - 실업/기술학교 설립
  - 지계 발급 (근대적 토지 소유권 확립 시도)

** 구본신참: 옛것을 근본으로 새로운 것을 받아들임 (=동도서기)

### 러일전쟁 (1904~1905년)
: 일본이 러시아 공격해 승리  
+ 한/일 의정서  
  : 전쟁 시 필요한 한국의 군사적 요충지 마음껏 이용 가능 
+ 제1차 한/일 협약
  - 메가타 경제고문: 화폐정리사업 주도
  - 스티븐스 외교고문: 조선의 식민지화 정당성 홍보  
  => 1909 샌프란시스코에서 사살 (장인환, 전명운)  

###  가쓰라/태프트 밀약 (1905년)
: 일본 총리 가쓰라 & 미국 대통령 특사 태프트 장관이 맺은 비밀 협약  
=> 미국-필리핀, 일본-조선 지배 승인 
+ 제2차 영/일 동맹 => 영국-인도, 일본-조선 지배 승인 
+ 포츠머스 강화 조약 => 러시아의 한반도 완전 철수

### 을사늑약 (1905년)
: 이토 히로부미가 강요한 제2차 한/일 협약 => 외교권 박탈
+ 을사오적
  - 이완용, 이지용, 이근택, 권중현, 박제순
+ 통감부 설치 
  - 초대 통감: 이토 히로부미
  - 통감정치 => 내정간섭 심화
+ 비판/반대
  - 시일야방성대곡 
    * 황성신문에 실린 장지연의 논설 
    * = 오늘 목 놓아 통곡하노라
    * 이 개, 돼지보다 못한 것들아
  - 민영환 자결
  - 오적 암살단
    * 나철, 오기호
  - 을사의병
    * 최익현 태인에서 봉기
    * 신돌석 (최초 평민 의병장)
