# 한국사 고급 (조선후기~개항기)

# 조선 후기

## 조선후기의 정치
* 비변사, 5군영, 속오군
* 예송, 환국
* 영조, 정조
* 세도정치, 삼정의 문란, 홍경래의 난, 임술 농민 봉기
* 영정법, 대동법, 균역법
* 광작, 도조법, 선대제, 덕대
* 공인, 통공 정책, 송상, 상평통보
* 공명첩, 잔반, 소청 운동, 노비종모법, 향전

## 조선후기의 사회
* 실학, 균전론, 한전론, 여전론
* 소비론, 홍대용

* 한글소설, 양반전, 청화백자
* 법주사 팔상전, 진경산수화, 풍속화, 민화
* 정감록, 서학, 동학 

# 개항기

## 흥선대원군 ~ 갑신정변
* 비변사 폐지, 경복궁 중건, 서원 철폐, 호포제, 사창제
* 병인박해, 병인양요, 오페르트 도굴 미수 사건
* 제너럴 셔먼호 사건, 신미양요
* 강화도 조약, 최익현
* 통리기무아문, 박문국, 조선책략, 조사 시찰단, 보빙사
* 위정척사 운동, 임오군란, 제물포 조약
* 개화파, 갑신정변, 톈진 조약, 지조법 개혁
* 영국 거문도 점령

### 흥선 대원군의 개혁정치
+ 비변사 폐지  
:  행정->의금부, 군사 -> 삼군부 => 세도가문 타파
+ 법규정비  
: 대전회통 (경국대전 + 속대전)
+ 경복궁 중건
	- 원납전 징수
	- 당백전 발행 (상평통보 * 100) => 인플레이션
	- 부역 동원
	- 양반 묘지림 벌목   
+ 서원철폐  
: 47개소만 남기고 철폐 => 지방 권력/제사비용 부담 제거   
ex) 만동묘 철폐 (명나라 황제 신종) 
+ 경제정책
	- 양전사업: 토지조사사업 (면세지 색출)
	- 호포법:  호마다 포 제출 (양반 포함)
	- 사창제: 민간인에게 빈민구휼창고 관리를 위임   
	=> 환곡의 문란 시정(공무원의 이자착취 방지)

** 운현궁에 거주 (개인 저택이었으나 고종 즉위 후 궁 칭호)  
** 진실로 백성에게 해되는 것이 있으면 비록 공자가 다시 살아난다 하더라도 나는 용서하지 않겠다.

### 쇄국정책 & 문호개방
+ **사건**
  - 병인박해  
    : 프랑스로 러시아 견제하려다 유생들 반대로 천주교 박해 
  - 제너럴 셔먼호 사건
    * 평양에서 통상 요구 및 난동 부린 미국 상선을 박규수가 불태움 
  - 병인양요 (1866년) 
    * 프랑스의 강화도 공격    
      => 한성근(문수산성), 양헌수(정족산성) 반격  
    * 외규장각 약탈  
      ex) 직지심체요절, 왕오천축국전
  - 오페르트 도굴 미수 사건 (1868년)  
    * 독일이 남연군(흥선대원군 아버지) 묘 도굴해 통상요구 계획   
     => 다층 석회 + 충청도 덕산 주민들 저항   
	 =>  실패  
  - 신미양요 (1871년)  
    * 미국 로저스 제독의 강화도 공격  
     => 어재연 장군 방어 (초지진, 덕진진 함락 -> 광성보 항전) ???   
    * 척화비 건립  
     => 통상수교 거부정책   
     => 서양 오랑캐들과 사이좋게 지내자고 주장하는 것은 매국하자는 것이다
  - 운요호 사건
  - 강화도 조약  
    * = 조/일 수호 조규  
      (청-조선 관계 단절 위해 자주국 칭호)
    * 민씨 정권 및 통상개화파가 수교 결단  
      ex) 박규수, 유홍기, 오경석 
    * 최초의 근대적 (국가vs국가) 불평등 조약  
       => 부산/원산/인천 개항  
       => 해안측량권, 치외법권(영사 재판권)  
  - 조/미 수호 통상 조약
    * 거중조정
    * 관세
    * 최혜국 대우
    * 최초의 대 서양 조약   
  - 조/프 수호 통상 조약
    * 천주교 포교 허가
  - 임오군란 (1882년)
    * 개항 후 곡물가격 폭등 + 월급 밀림 -> 2영의 구식 군인과 도시 빈민이 민란
    * 선혜청(1경당 12두 대동미 관리) 공격 -> 관리자 민겸호 살해
    * 흥선대원군 정계 복귀 (약 1개월)
    * 명성황후 충주로 피신 -> 청에 도움 요청해 진압 -> 흥선대원군 톈진으로 압송   
  - 제물포 조약 (1882년)
    * 임오군란 때 일본 교관/공사 공격받음 => 군대 주둔권 및 배상금
  - 조/청 상민 수륙 무역 장정
    * 청나라 상인이 내지까지 들어와 상행위 가능 
  - 갑신정변 (1884년)
    * 김옥균, 박영효, 홍영식, 서재필 등  
       => 홍영식 사망, 나머지는 일본 망명
    * 우정국 개국 축하연에서 민씨 정권 실세 제거
    * 고종 & 명성 황후 경운궁으로 대피 -> 창덕궁으로 복귀 -> 청군개입으로 종료
    * 14개조 개혁안 발표   
      - 내각 중심 입헌군주제
      - 규장각 혁파, 혜상공국(보부상 관리) 폐지  
	  - 재정의 호조 일원화, 지조법(토지세금) 개혁
	  - 신분제 폐지 
  - 한성조약
    * 갑신정변 후, 배상금 및 일본 공사관 신축 비용 요구 
  - 톈진조약
    * 갑신정변 후, 청/일 군대 동시 철병
    * 군대 개입 시 상호 통보 약속 
  - 거문도 점령 (1885년)
    * 청 견제 위해 러시아와 밀약 체결 시도 -> 영국이 견제 후 거문도 점령
    * 한반도 중립화론 제기 (유길준 & 독일 영사 부들러)

+ **세력**
  - 최익현  
    * 왜양일체론 개항반대운동 (강화도 조약 시)
    * 흥선 대원군 탄핵 상소 (고종 20대)
    * 을사늑약 심경 토로 포고문 (1905년) 
    * 항일투쟁 호소 및 철도/일본 불매
    * 을사의병 전개 (전북 태인 -> 순창 진격)
    * 쓰시마 섬 유배 (일본 제공 음식 거절하고 순국) 
  - 위정척사운동
    * 바른 것을 지키고 사악한 것을 물리치자는 유교적 운동

    | 연도 |활동|대표인물|
    |---|---|---|
    | 1860년대|통상수교반대, 척화주전론| 이항로 척화상소, 기정진|  
    | 1870년대|개항반대, 왜양일체론|최익현|  
    | 1880년대|개화반대, 영남 만인소|이만손|  
    | 1890년대|의병투쟁 본격화|유인석, 이소응 의병장|  

  - 개화파
    * 김옥균 등 개화 주장하던 정치세력
    * 문명 개화론에 입각한 급진적 개화 표방  
      ex) 일본 메이지 유신 
    * 청 고문 묄렌도르프가 화폐 발행 시도 -> 인플레 이유로 반대 
    * 일본 차관 주장 -> 수구파 방해로 차관 실패 -> 정치적 위기 

+ **근대 기관**
  - 통리기무아문 (1880년)  
    * 개화정책 총괄 관청  
      : 교린사/군무사/통상사 등 12사 총괄
    * 사절단 파견  
      ex) 조사시찰단, 영선사  
    * 중앙군 개편    
      => 5군영 -> 2영 (무위영+장어영)   
      => 별기군 설치 (신식군대) 
  - 박문국 (1883년)
    * 근대적 인쇄 기술 도입
    * 한성순보 발간 (한국 최초 신문, 발행주기:10일) 
  - 전환국 (1883년)  
    : 근대 화폐 발행 
  - 기기창 (1883년)  
    : 근대 무기 제조 
  - 우정국 (1884년)  
    : 근대 우편 업무 기관 

+ **해외 사절단**
  - 일본 조사 시찰단 (신사유람단) 
    * 박정양 등
    * 암행어사 형태로 비공개 파견 (유생 등 반발 우려) 
  - 일본 수신사 (1876년)
    * 1차: 김기수 등, 일본 관청/군사학교 시찰
    * 2차: 김홍집 등, 조선책략(친중국/결일본/연미국 조언) 입수
  - 청 영선사 (1881~82년)
    * 김윤식 등 톈진에 파견
    * 과학/무기 기술 및 외국어 학습 -> 기기창 설립
    * 비용 부족으로 중도 귀국
  - 미국 보빙사 (1883년)
    * 전권대신 민영익, 홍영식, 서광범 등
    * 유길준은 그대로 유학 => 서유견문 집필
    * 근대 시설 시찰 -> 우정국 / 육영공원 설치
    * 일화: 아서 대통령 접견 시 엎드려 절함

### 동학농민운동
+ 교조 신원 운동
  - 창시자 최제우의 명예 회복 요구
  - 1892년 전라도 삼례 집회  
  - 1893년 보은 집회  
    : 탐관오리 숙청 및 외세 반대 구호 등장 => 정치/사회 운동으로 발전 
+ 고부 농민 봉기 (1894년)
  - 전라도 고부 군수 조병갑 -> 학정 및 만석보 축조 
  - 녹두 장군 전봉준
+ 제1차 농민 봉기
  - 안핵사 이용태가 고부 봉기 주동자 체포
  - 백산 집결 -> 4대 강령 및 격문 발표  
    : 보국안민, 제폭구민 
  - 정읍 황토현 전투 승리
  - 장성 황룡촌 전투 승리
  - 전주성 함락 -> 청/일본 군대 파견 -> 전주화약 
+ 전주화약   
  - 집강소(농민자치기구)의 폐정 개혁안 12개조 단행
    * 토지 분재
    * 탐관오리 처단
    * 신분제 폐지
    * 과부 재가 허용 
  - 정부의 교정청 설치 -> 개혁요구 수용 
+ 제2차 농민 봉기
  - 청일전쟁 발발 -> 교정청 페지 및 군국기무처 설립 -> 농민 봉기
  - 전봉준 납접 & 손병희 북접 연합
  - 공주 우금치 전투 (1894년 11월)
  - 순창에서 전봉준 체포  
+ 사료
  - 사발통문  
   : 주동자 모름 -> 공동책임 
  - 장태  
   : 닭 둥지에 볏집 넣고 굴리는 무기
### 갑오/을미개혁
+ 제1차 갑오개혁 (1894년)

  - 정치
    * 1차 내각: 온건 개화파 김홍집 중심
    * 개국 기년 사용: 1392년 조선 건국  (청 연호 폐지)
    * 궁내부 설치: 왕실 사무 관리 (왕실-국가 분리)

    * 도량형 통일
    * 재정의 탁지아문 일원화
    * 조세 금납화

  - 사회
    * 공/사 노비법 혁파 -> 신분제 폐지
    * 연좌제 폐지
    * 조혼 금지
    * 과부 재가 허용

### 아관파천 (1896년)
+  명성왕후 시해 후, 러시아 공사관으로 이동한 고종이 일본 주도 개혁 무효화
	- 단발령 비강제화
	-  김홍집 처단(갑오/을미개혁 진행)
+  삼림 채벌권 요구 => 최혜국 대우 적용 확장
	-  미국: 운산 금강 채굴권 요구
	-  프랑스: 광산채굴권 요구
+ 부산 절영도 조차 요구 
	- 석탄 공급 용 부동항 
	- 독립협회 만민 공동회가 무산 시킴

### 독립협회
: 서재필/이상재 등 독립신문 간행을 위해 모인 사람들이 설립 (1896년부터 활동 활발)
+ 고종 환궁 주장
+ 의회설치 주장
+ 독립문 건설 
	- 영은문 자리에 건립
	-  청나라로부터 독립 다짐
+ 민중 계몽 활동
	- 토론회, 강연회
+ 정치 논의 집회
	- 만민공동회: 종로 종각 앞 연설 & 논의   
	  => 절영도 조차 무산, 한/러 은행 철수, 프랑스 광산 채굴권 방어
	- 관민 공동회 : 정부 관료도 참여 (박정양 내각~)

### 환구단 (1897년)

+ 국가의 주권 -> 황제에게


### 관민공동회

### 황국협회

+ 고종이 군대 동원해 독립협회 강제해산 (1898년)

### 대한국 국제 (1899년)
: 황제의 천상천하 유아독존 최고권력 선포한 헌법의 일종
+ 제2조
  - 대한 제국의 정치는 이전부터 이후까지 변하지 않을 전제 정치이다
+ 원수부 설치
  - 모든 군사력 황제에게 일원화
+ 광무개혁 본격화
  - 상공업 진흥
  - 실업/기술학교 설립
  - 지계 발급 (근대적 토지 소유권 확립 시도)

** 구본신참: 옛것을 근본으로 새로운 것을 받아들임 (=동도서기)

### 러일전쟁 (1904~1905년)
: 일본이 러시아 공격해 승리  
+ 한/일 의정서  
  : 전쟁 시 필요한 한국의 군사적 요충지 마음껏 이용 가능 
+ 제1차 한/일 협약
  - 메가타 경제고문: 화폐정리사업 주도
  - 스티븐스 외교고문: 조선의 식민지화 정당성 홍보  
  => 1909 샌프란시스코에서 사살 (장인환, 전명운)  

###  가쓰라/태프트 밀약 (1905년)
: 일본 총리 가쓰라 & 미국 대통령 특사 태프트 장관이 맺은 비밀 협약  
=> 미국-필리핀, 일본-조선 지배 승인 
+ 제2차 영/일 동맹 => 영국-인도, 일본-조선 지배 승인 
+ 포츠머스 강화 조약 => 러시아의 한반도 완전 철수

### 을사늑약 (1905년)

  - 민영환 자결

    * 나철, 오기호
  - 을사의병

    * 신돌석 (최초 평민 의병장)
